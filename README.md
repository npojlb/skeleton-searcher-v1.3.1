# Skeleton Searcher 1.3.1

适用于 Minecraft Java Edition 1.21.11 + Fabric 的纯客户端区域高亮模组。

## 核心规则

- 只能使用共同交集运算。
- 第 1、2 个球体只记录并计算，不进行任何世界渲染。
- 从第 3 个球体开始，显示所有已记录球体的共同交集。
- 新球体加入后若共同交集为空，则拒绝添加并提示“已超出区域”。
- 球壳范围：250–150、150–100、100–50、50–25、50–0。

## 渲染样式

设置页面可在以下两种样式之间切换：

- 球壳：只显示最终交集区域的边界方块。
- 半透明方块：显示最终交集区域内的半透明白色方块。

最大本地渲染距离为 300 格。极大范围会自动进行稀疏采样，以避免客户端长时间卡死。

## 快捷键

- `O`：打开中文设置页面。
- `Delete`：清除全部球体。

快捷键可在“选项 → 控制 → 按键绑定 → Skeleton Searcher”中修改。

## 配置文件

```text
.minecraft/config/skeleton_searcher.json
```

## 构建

项目使用 Java 21、Gradle 9.2.0、Fabric Loom 1.14.10。GitHub Actions 构建成功后，在运行记录的 Summary 页面底部下载 Artifact。

## 第四版变化

当前半径档位及界面文字颜色：

- `250–150`：浅红
- `150–100`：深蓝
- `100–50`：黄色
- `50–25`：亮绿色
- `50–0`：亮蓝色

两种渲染样式都会忽略地形深度遮挡，因此可透过墙体和地面观察高亮结果。每次打开设置页面时，X、Y、Z 默认填入玩家当前所在方块的坐标；仍可手动编辑或点击“使用玩家坐标”刷新。
